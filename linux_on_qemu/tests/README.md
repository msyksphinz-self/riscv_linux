# テストプログラム

このディレクトリには、QEMU上で実行するテストプログラムが含まれています。

## テストプログラムの追加方法

1. 新しいCプログラムを`tests/`ディレクトリに追加します
2. `tests/Makefile`の`TESTS`変数にプログラム名を追加します
3. Dockerイメージを再ビルドすると、自動的にコンパイルされ、`testfs.img`に含まれます

## 実行方法

QEMU起動後、以下のコマンドでテストファイルシステムをマウントできます：

```bash
mount -t ext2 /dev/vdb /mnt/testfs
```

テストプログラムは`/mnt/testfs/bin/`に配置されています。

## 現在のテストプログラム

- `hello`: シンプルなHello Worldプログラム
- `test_syscall`: システムコールのテストプログラム
